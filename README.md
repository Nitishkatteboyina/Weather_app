# Weather App

A simple weather app to check the current weather and forecast.

### Features

* Current weather conditions
* 5-day forecast
* Search for weather by location

### Built With

* HTML/CSS
* JavaScript

### Getting Started

Clone the repository and open `index.html` in your browser.
